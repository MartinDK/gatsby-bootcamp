This website began from watching Andrew Mead's tutorial [Gatsby JS - The Great Gatsby Bootcamp](https://youtu.be/8t0vNu2fCCM?t=9200). I've modified the project from the tutorial to create this website, with help from [GatsbyJS.org](https://www.gatsbyjs.org/) documentation.


### MDX files
Blog and project content on this website is created in [MDX](https://mdxjs.com) files. MDX files are similar to Markdown, but with the ability to add JSX components. The files are pushed via Github to Netlify where they are processed and turned into a webpages. I mostly edit the MDX files with [Working Copy](https://workingcopyapp.com) on an iPad.
